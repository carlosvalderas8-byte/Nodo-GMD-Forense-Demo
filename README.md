# 🛡️ GMD 2026 - Nodo de Seguridad Blindado
**Tecnología de Cifrado AES-256 Local con Auditoría Forense.**

Este repositorio contiene la unidad de demostración del sistema **GMD 2026**. Un software diseñado para entornos donde la privacidad es crítica y la nube no es una opción segura.

### 🔒 Características Principales:
* **Blindaje por Hardware (HWID):** La licencia se vincula físicamente a la placa base del equipo. No es transferible.
* **Cero Nube:** El sistema funciona 100% offline. Tus datos nunca salen de tu perímetro.
* **Algoritmo de 32 rondas:** Protección de nivel militar contra ataques de fuerza bruta.

### 🚀 Instrucciones para la Demo:
1. Descarga el archivo ejecutable `GMD_2026.bat` (disponible en la raíz del proyecto).
2. Ejecútalo en el equipo donde desees instalar el nodo.
3. El sistema generará un **ID de HARDWARE** único.
4. Envía este ID al administrador para validar tu prueba o activar tu licencia vitalicia.
